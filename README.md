How to use RTrain

**Put this in your Gemfile**
```
gem 'rtrain', '~> 0.1.5'
```
Then `bundle install`

**After generating a scaffold, enter this into the command line:**

```
rails generate rtrain:install
```
**That's it! Now your Rails application is equipped with an RTrain!**
